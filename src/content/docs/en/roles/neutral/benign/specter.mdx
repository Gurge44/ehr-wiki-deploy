## Role Description

The **Specter** is a Neutral Benign role. The Specter must complete their tasks, and also must be dead by the end of the game. If both of these requirements are met, the Specter wins with the winning team. The Specter is allowed to do tasks while alive. The Specter may also win alone if configured to do so.

## Role Options

<details>
<summary>
### Max
</summary>
**Type:** `Number`\
**Default:** `1`

Specifies how many Specters can appear in one game.
</details>

<details>
<summary>
### Can Vent
</summary>
**Type:** `Toggle`\
**Default:** `OFF`

Decides whether or not the Specter can Vent.
</details>

<details>
<summary>
### Snatches Victory
</summary>
**Type:** `Toggle`\
**Default:** `OFF`

Decides whether or not the Specter wins alone after finishing tasks instead of with the winning team.
</details>

<details>
<summary>
### Can Guess In Guesser Mode
</summary>
**Type:** `Toggle`\
**Default:** `OFF`

Decides whether or not the Specter can Guess in Guesser mode.
</details>

<details>
<summary>
### Override Specter's Tasks
</summary>
**Type:** `Toggle`\
**Default:** `OFF`

Decides whether or not the Specter gets a custom amount of tasks set by the host.

<summary>
#### Specter has Common Tasks

Decides whether or not the Specter gets Common Tasks.
</details>
