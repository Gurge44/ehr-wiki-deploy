# Endless Host Roles Documentation

Welcome to the official Wiki for **Endless Host Roles**, a powerful mod for **Among Us** that adds host-controlled role customization, game modes, and advanced configuration options.

Whether you're a new host or a seasoned modder, this site will help you understand how to install, configure, and get the most out of the mod.

---


## 🚀 Features

- 🎭 Over **400 custom roles** with unique abilities
- 🧠 Dynamic **game modes** like Speedrun, Quiz, and Stop And Go
- 🛠️ Deep customization for hosts (role weights, rules, modifiers)
- 🔁 **Auto-updating** config system
- 🧩 Works seamlessly with **BepInEx** and other mods
