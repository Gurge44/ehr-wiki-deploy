This page provides a structured overview of all available keybinds in **Endless Host Roles**. Keybinds are grouped by where they are used (lobby, in-game, or debug mode) and by which users can access them (hosts, modded clients, or developers). Use this as a quick reference to master the controls and get the most out of your experience.

---

## Usable In Lobby

<div class="responsive-card-table keybind-table-wrapper">
<div class="keybind-table-wrapper">

| Action                                                 | Key     | Access                                             |
|--------------------------------------------------------|---------|----------------------------------------------------|
| Cycle to the next page of the settings viewer          | `Tab`   | <span class="modded-clients">Modded Clients</span> |
| Jump to settings page 1–9                              | `1`–`9` | <span class="modded-clients">Modded Clients</span> |
| Instantly start the game (skip countdown)              | `LSHIFT`| <span class="host-only">Host Only</span>           |
| Cancel the countdown                                   | `C`     | <span class="host-only">Host Only</span>           |

</div>
</div>
