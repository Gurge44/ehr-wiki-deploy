## ‚è±Ô∏è Auto Hosting & Timers

:::note
The settings below are only applicable when Auto Host is enabled in [Client Options](../client-options). If Auto Host is disabled, these timers and thresholds will not take effect.
:::

<details>
  <summary>Minutes to wait before auto-starting</summary>

  * **Type**: Time (minutes)
  * **Description**: Auto-starts after this delay if minimum player threshold is met.
  * **Default**: `2.0`
</details>

<details>
  <summary>Force start when Lobby Timer goes below</summary>

  * **Type**: Time (minutes)
  * **Description**: Forces game start regardless of player count once this time limit is hit.
  * **Default**: `6.0`
</details>

<details>
  <summary>Minimum Player Threshold to auto-start</summary>

  * **Type**: Number
  * **Description**: Required players to enable auto-start.
  * **Default**: `5`
</details>

<details>
  <summary>Initial Countdown for auto-starting</summary>

  * **Type**: Time (seconds)
  * **Description**: Countdown duration before the game starts once auto-start is triggered.
  * **Default**: `20`
</details>
